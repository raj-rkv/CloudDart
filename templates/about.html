About us

CloudDart is a cloud Web API which will interact with different Cloud service provider (Azure, Amazon Web Services and Google Cloud Platform) to give the user a common interface so they can compare and use the services of different cloud platform easily with simple or handy interface. In this API user will be able to run the different services of cloud in any platform (Azure, AWS and GCP) at a single place.
